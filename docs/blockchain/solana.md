# Solana 开发

## 1. 安装 sol-cli

1. 安装下载器

   ```shell
   sh -c "$(curl -sSfL https://release.solana.com/v1.18.4/install)"
   ```

2. 更新环境变量，注意版本变化

   ```shell
   echo 'export PATH="/root/.local/share/solana/install/active_release/bin:$PATH"' >> ~/.zshrc                                                               ‹system›
   source ~/.zshrc
   ```
